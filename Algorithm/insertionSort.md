# 삽입정렬 알고리즘

## 문제

정수를 요소로 갖는 배열을 입력받아 오름차순으로 정렬하여 리턴해야 한다.

## 입출력 예시

```javascript
let output = insertionSort([3, 1, 21]);
console.log(output); // --> [1, 3, 21]
```

```javascript
const insertionSort = function (arr) {
  // TODO: 여기에 코드를 작성합니다.
  /*
  삽입정렬이란? [3,7,2,5,1,4]
  삽입 정렬은 왼쪽에서 오른쪽으로 가면서 각 요소들을 왼쪽 요소들과 비교하여 알맞은 자리에 삽입하는 형식의 정렬 방법이다. 
  항상 두번째 요소를 왼쪽 요소와 비교하면서 시작한다.
  1회전에서는 왼쪽 요소보다 두번째 요소가 크기 때문에 아무런 일이 일어나지않는다. 3 < 7
  2회전에서는 2를(arr[2]) 왼쪽 두개의 데이터와 비교하여 가장 가까운 왼쪽에 끼워넣는다. 3,7 < 2 -> 2,3,7
  3 회전에서는 5를 왼쪽 데이터와 비교하여 3과 7사이에 끼워넣는다. 2,3,7 < 5 -> 5 < 7 -> 2,3,5,7
  */

  for(let i = 1; i < arr.length; i++){ // arr = [3,7,2,5,1,4]
    let cur = arr[i]; // 1회차 -> i = 1, cur = 7, 2회차 -> i = 2, cur = 2
    let left = i - 1  // 1회차 -> left = 0, 2회차 -> left = 1

    while(left >= 0 && arr[left] > cur){ // 1회차 -> 3 >= 0 &&  3 > 7   // 2회차 -> 1 >= 0, 7 > 2
      arr[left + 1] = arr[left] // arr[2] = arr[1] -> 2 = 7 // 바꿔 주는 과정
      left --; 
    }
    arr[left + 1] = cur  // cur = 7
  }
  return arr
};

```
